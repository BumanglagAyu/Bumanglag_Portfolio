## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Create an environment file

Create a `.env.local` file in the root directory and add:

```env
NEXT_PUBLIC_SITE_URL=http://localhost:3000
```

If using the contact form, also add your Formspree endpoint:

```env
CONTACT_FORM_ACTION_URL=https://formspree.io/f/your-form-id
```

### 3. Run the development server

```bash
npm run dev
```

Open the project in your browser:

```txt
http://localhost:3000
```

## Deployment

This portfolio can be deployed using **Vercel**.

Before deploying, add the following environment variables in your Vercel project settings:

```env
NEXT_PUBLIC_SITE_URL=https://your-deployed-site-url.vercel.app
CONTACT_FORM_ACTION_URL=https://formspree.io/f/your-form-id
```

After deployment, update `NEXT_PUBLIC_SITE_URL` to match the final live URL.
